# Галерея Фільмів 

Проект розроблений за допомогою **React**.

## Опис

Застосунок являє собою невелику галерею кінофільмів, яка реалізовує базові концепції React:
розбиття на компоненти, створення семантичної структури та динамічне виведення масивів за допомогою методу `.map()`.

### Головні особливості:
- **Семантична структура**: присутні теги `<header>`, `<main>`, `<footer>`.
- **Власні компоненти**: застосунок створює та групує компоненти (Header, Footer, MovieList, MovieCard).
- **Стилізація**: використані CSS-змінні для кольорів та створено адаптивний дизайн із приємними візуальними ефектами (наведення та тіні).
- **Опис контенту**: постери відомих фільмів, їх рейтинг та короткі описи українською мовою.

## Запуск проекту

1. Спочатку завантажте залежності:
   ```bash
   npm install
   ```
2. Запустіть локальний сервер:
   ```bash
   npm run dev
   ```
